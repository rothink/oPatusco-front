# Passo a passo front

- Clonar repositório
  >
        git clone {.git-repo}


- Entrar no repositório clonado
  >
        cd {folder}


- Buildar e subir os serviços com docker
  >
        make install

- Rodar os testes
  >
        make test

- go to page
  >
        http://172.24.0.2:3000/
